# 随机万事屋 （Random Utility Hub）

一站式随机工具集合，帮助你在日常决策、娱乐互动与轻量抽签场景中快速得到结果。

## 功能概览

- 🎲 随机数生成（区间可配置）
- 🎡 选择困难助手（随机转盘）
- 🔮 今日运势（多维度评分）
- 🎁 抽奖工具（名单抽取）
- 🪙 抛硬币
- 🎯 掷骰子
- 👥 随机分组
- 📜 历史记录查看

## 产品特性

- 🌐 中英文双语切换
- 🌓 深色 / 浅色主题切换（自动记忆偏好）
- 📱 响应式界面，兼容桌面与移动端

## 技术栈

- Next.js 15（App Router）
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React

## 本地开发

```bash
npm install
npm run dev
```

默认访问地址：`http://localhost:3000`

## 质量校验

```bash
npm run lint
npm run build
```

## 目录说明

- `src/app`：页面与根布局
- `src/components`：业务组件与 UI 组件
- `src/lib`：工具函数
- `public`：静态资源（包含网站图标等）
